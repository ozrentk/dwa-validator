1. Install SQL Server Express LocalDB if it is not already installed
 https://learn.microsoft.com/en-us/sql/database-engine/configure-windows/sql-server-express-localdb

2. Unzip DwaValidatorApp.zip on your PC

3. Open the .dwavalidator file in a text editor and check if it matches 
   the location where you want the validation archive (StorageRootFolder) to be
   unpacked locally. If it doesn't fit your need, change as desired and save file.

4. Run DwaValidatorApp.exe

5. Using drag-and-drop, drag your project archive into the application

6. Start validation by double-clicking on it
